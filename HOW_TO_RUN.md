# วิธีรัน Inhouse Insider

## ต้องการ
- Node.js (ดาวน์โหลดที่ https://nodejs.org)

## ขั้นตอน

### 1. เปิด Terminal แล้วไปที่โฟลเดอร์นี้
```
cd "path/to/Inhouse Insider"
```

### 2. ติดตั้ง dependencies (ครั้งแรกครั้งเดียว)
```
npm install
```

### 3. รันเซิร์ฟเวอร์
```
npm start
```

### 4. หา IP ของเครื่อง
- **Mac:** System Settings → WiFi → Details → IP Address
- **Windows:** `ipconfig` ใน Command Prompt

### 5. แชร์ลิงก์ให้เพื่อน
เพื่อนพิมพ์ใน browser มือถือว่า:
```
http://[IP ของคุณ]:3000
```
ตัวอย่าง: `http://192.168.1.42:3000`

> ทุกคนต้องอยู่ใน WiFi เดียวกัน

---

## วิธีเล่น

1. **Host** สร้าง Room → ได้รหัส 4 ตัวอักษร
2. **เพื่อนๆ** พิมพ์รหัสเข้าร่วม (ต้องการอย่างน้อย 3 คน)
3. Host กด **Start Game**
4. แต่ละคนดู **role ลับ** ของตัวเอง:
   - 👑 **Master** — รู้คำลับ ตอบคำถาม YES/NO/IDK
   - 🕵️ **Insider** — รู้คำลับลับๆ ช่วย guide โดยไม่ให้ถูกจับ
   - 🙋 **Common** — ไม่รู้คำ ถามคำถามเพื่อทาย
5. มีเวลา **3 นาที** ถามตอบ
6. เมื่อทายคำได้ (หรือหมดเวลา) → โหวตว่าใครคือ Insider
7. จับ Insider ได้ → Common ชนะ / จับไม่ได้ → Insider ชนะ
